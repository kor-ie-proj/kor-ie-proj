# 경제지표 예측 모델링

## 개요

LSTM(Long Short-Term Memory) 신경망을 활용하여 한국의 주요 경제지표를 예측하는 시계열 예측 모델입니다. 한국은행 경제통계시스템(ECOS)에서 수집한 경제지표 중 예측력이 높은 핵심지표들을 선별하여 미래 경제 동향을 예측합니다.

## 예측 대상 지표

### 선정 기준
경제 예측에서 가장 중요한 선행성, 포괄성, 실시간성을 기준으로 핵심지표를 선정했습니다.

### 핵심 경제 지표

1. **경기선행지수 (leading_index)**
   - 역할: 가장 포괄적인 공식 선행지표
   - 특징: 경기 전환점을 3-6개월 앞서 예측
   - 중요도: 정부 정책 수립의 핵심 참고지표

2. **장단기 금리차 (term_spread)**
   - 역할: 경기 전환점을 예측하는 가장 강력한 시장 기반 지표
   - 계산: 국고채 10년 - 국고채 3년 금리
   - 특징: 경기침체 예측에서 높은 정확도

3. **신용스프레드 (credit_spread)**
   - 역할: 금융 스트레스와 신용 여건을 측정하는 실시간 지표
   - 계산: 회사채 BBB- - 국고채 3년 금리
   - 특징: 금융위기 조기경보 기능

4. **경제심리지수 (esi)**
   - 역할: 소비와 투자를 모두 선행하는 가장 포괄적인 민간 부문 심리 지표
   - 특징: 실물경제 동향을 반영하는 종합지표
   - 범위: 소비자, 기업가 심리를 통합 측정

5. **원달러 환율 (exchange_usd_krw)**
   - 역할: 대외 부문의 영향을 포착하는 개방경제 지표
   - 특징: 수출입, 자본유출입 동향 반영
   - 중요도: 한국경제의 대외의존도를 고려한 필수지표

6. **소비자물가지수 (cpi)**
   - 역할: 물가 상승 동력을 반영
   - 특징: 월별 변화율로 단기 물가 압력 측정
   - 활용: 통화정책 방향성 예측

## 기술적 특징

### 모델 아키텍처
- **모델 유형**: LSTM (Long Short-Term Memory) 신경망
- **구조**: 다층 LSTM + Dense 레이어
- **시퀀스 길이**: 12개월 (1년간의 과거 데이터 활용)
- **출력**: 다중 지표 동시 예측

### 상세 구조
```
Input Layer: (12, 6) - 12개월 × 6개 지표
LSTM Layer 1: 100 units, return_sequences=True
Dropout: 0.2
LSTM Layer 2: 100 units, return_sequences=True  
Dropout: 0.2
LSTM Layer 3: 50 units, return_sequences=False
Dropout: 0.2
Dense Layer 1: 25 units, ReLU activation
Dense Layer 2: 6 units (출력층)
```

### 데이터 전처리
- **정규화**: MinMaxScaler (0-1 범위)
- **시퀀스 생성**: 슬라이딩 윈도우 방식
- **결측값 처리**: 선형 보간법
- **데이터 분할**: 훈련 80%, 테스트 20%

### 훈련 설정
- **옵티마이저**: Adam (learning_rate=0.001)
- **손실함수**: Mean Squared Error
- **평가지표**: Mean Absolute Error
- **에포크**: 최대 100 (Early Stopping 적용)
- **배치 크기**: 32
- **검증 분할**: 20%

### 콜백 함수
- **EarlyStopping**: patience=20, 과적합 방지
- **ReduceLROnPlateau**: patience=10, 학습률 자동 조정

## 성능 평가

### 평가 지표
- **RMSE (Root Mean Square Error)**: 예측 오차의 크기
- **MAE (Mean Absolute Error)**: 절대 오차의 평균
- **R² (결정계수)**: 설명력 (1에 가까울수록 높은 예측력)
- **상관계수**: 실제값과 예측값 간의 선형 관계

### 시각화 결과
1. **훈련 과정**: Loss와 MAE의 에포크별 변화
2. **예측 비교**: 실제값 vs 예측값 시계열 그래프
3. **상관관계**: 산점도와 상관계수 분석
4. **미래 예측**: 과거 데이터와 미래 예측의 연결 시각화
